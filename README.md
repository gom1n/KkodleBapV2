# 🍚 꼬들밥(KkodleBap) 🍚

![logo](./assets/logo.png)

한글 자모 6개로 정답을 맞히는 게임, **꼬들밥!**  
귀여운 고봉밥과 함께 한글 자모 게임의 매력에 빠져보세요.  
오늘의 꼬들밥, 당신은 찾을 수 있을까요?
